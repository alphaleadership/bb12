Commandes

Description

Les commandes de modération permettent de ban, kick ou encore mute une personne donnée. Les commandes de modérations sont utilisables par les personnes ayant les permissions sur votre serveur Discord (exemple: permission de kick pour utiliser la commande kick)

Cette commande permet de bannir une personne de votre serveur.
Utilisation: {prefix}ban <@utilisateur> <raison>

Cette commande permet d'expulser une personne de votre serveur.
Utilisation: {prefix}kick <@utilisateur> <raison>

Cette commande permet de mute une personne de votre serveur.
Utilisation: {prefix}mute <@utilisateur> <raison>
Vous pouvez aussi démute une personne avec la commande suivante:
Utilisation: {prefix}unmute <@utilisateur> <raison>

Cette commande permet d'ajouter un warn à une personne
Utilisation: {prefix}warn <@utilisateur> <raison>
Vous pouvez voir tous les warns d'une personne avec la commande warns:
Utilisation: {prefix}warns <@utilisateur>
Vous pouvez effacer un warn d'une personne avec la commande:
Utilisation: {prefix}del-warn <@utilisateur> <numéro de warn>

Cette commande permet de clear un certain nombre de messages passés en paramètre
Utilisation: {prefix}clear <nombre>

Anti-insulte

Description

L'anti-insulte permet de rentrer des mots interdits dans votre serveur. Si un de ces mots est écrit dans un des messages envoyés par l'utilisateur d'un serveur, ce message sera automatiquement effacé en avertissant l'utilisateur.

Salon de logs

Description

Le salon de logs permet de recevoir des messages pour chaque action d'un utilisateur, que ce soit, pour la suppression d'un message, la modification, un kick, un ban et bien plus!
